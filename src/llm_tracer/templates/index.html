A LangSmith-like observability platform for your LangChain and LangGraph applications. Track LLM calls, tool usage, and debug your AI agents with ease.

Integrate SDK

Add tracing to your LangChain application.

# Install the SDK
pip install llm-tracer-sdk

# Configure in your code
import llm_tracer_sdk
from langchain_openai import ChatOpenAI

# Configure with your API key
llm_tracer_sdk.configure(
    api_key="lt-your-api-key",
    endpoint="http://localhost:8000"
)

# Wrap your LLM - that's it!
llm = llm_tracer_sdk.TrackerLLM(ChatOpenAI(model="gpt-4"))

# All calls are now traced automatically
response = llm.invoke("Hello, how are you?")
